## Overview

This project is a Streamlit application that enables product registration with Azure Cloud integration. The application provides a user-friendly interface for managing and registering products using Azure cloud services as the backend.

![](/assets/Screenshot_2026-02-17_16-47-11.png)
![](/assets/Screenshot_2026-02-17_17-08-19.png)

## Features

- Product registration interface
- Azure Cloud integration
- Data persistence in Azure
- Real-time form validation

## Requirements

- Python 3.8+
- Streamlit
- Azure SDK for Python

## Installation

```bash
pip install streamlit azure-storage-blob pymssql
```

## Usage

Run the application with:

```bash
streamlit run main.py
```
